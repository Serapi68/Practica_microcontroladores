# Pruebas con Microcontroladores

Pequeño repositorio con ejemplos de programación de microcontroladores en ARDUINO IDE y MICROPYTHHON.

##  Objetivos

- Aprender diversas librerias de sensores.
- Generación de proyectos básicos y de facil creación.
- Analisis de como funciona un sitema secuencial.

##  Proceso:

Se han usado microcontroladores como Arduino y ESP32, usando C++ en Arduino IDE y Micropython comparando las capacidades de cada microcontrolador, se demuestran proyectos de facil acople y descargables para su uso.

## Herramientas:
- Arduino IDE.
- Thonny IDE.
- C++.
- Micropython.
