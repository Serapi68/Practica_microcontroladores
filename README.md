# Pruebas con Microcontroladores

Pequeño repositorio con ejemplos de programación de microcontroladores en ARDUINO IDE y MICROPYTHON.

##  Objetivos

- Aprender diversas librerías de sensores.
- Generación de proyectos básicos y de fácil creación.
- Análisis de como funciona un sistema secuencial.

##  Proceso:

Se han usado microcontroladores como Arduino y ESP32, usando C++ en Arduino IDE y Micropython comparando las capacidades de cada microcontrolador, se demuestran proyectos de fácil acoplamiento y descargables para su uso.

## Herramientas:
- Arduino IDE.
- Thonny IDE.
- C++.
- Micropython.
